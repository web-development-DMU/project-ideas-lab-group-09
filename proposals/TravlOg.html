TravlOg

Proposed by: elliott chijoke

Description

The idea of this site is for people wanting to travel for holiday, the sites aim will be for users to be able to travel the safest and cheapest way they can. In this site the user should be able to select their chosen destination and chosen budget, the site should then recommend things like best flights and available hotels in that destination. the user is then able to browse through the options provided and pick which one they prefer.

it needs a search engine and also a database that includes multiple holiday destinations, when the users searhes a prefered destination the data base is searched to find something that better fits their search
