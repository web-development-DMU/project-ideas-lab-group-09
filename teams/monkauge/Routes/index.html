Users

                    
    
                        
  • GET /users/:userId          →   returns individual user
    • 
                        
  • GET /users?query=username   →   returns filtered list of users
    • 
                        
  • POST /users                 →   creates new user record, redirects to GET /users
    • 
                        
  • POST /users/:userId         →   updates user record, redirects to GET /users
    • 
                        
  • POST /users/:userId/delete  →   deletes user record, redirects to GET /users
    •

Comments

                    
    
                        
  • GET /comments?query=userId      →   returns filtered list of comments by userId
    • 
                        
  • GET /comments?query=episodeId   →   returns filtered list of comments by episodeId
    • 
                        
  • GET /comments?query=characterId      →   returns filtered list of comments by characterId
    • 
                        
  • GET /comments?query=factionId      →   returns filtered list of comments by factionId
